# seduco_core

Empty project.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```
To run as live server:

```sh
npm start
```

## Running

```sh
node dist/bundle.js
```
